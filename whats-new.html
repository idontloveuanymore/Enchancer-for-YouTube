What's new in Enhancer for YouTube™ v2.0.122

The long-awaited update is finally here!
But don't expect new features, it's more a bugfix version...

R.I.P.

To comply with YouTube's Adblock Policy I had to remove all the “Ads Management”-related features (they caused issues anyway...). The good news is that you can still control the playback speed of ads, which is better than nothing ¯\_(ツ)_/¯
Install uBlock Origin if you want to block ads.

Improvements

  • Updated the appearence of the control bar for a better integration
  • Updated the YouTube-DeepDark theme (but removed the Red Light variant)
  • Miscellaneous code improvements for a better user experience

Bugs

  • Fixed broken features for embedded players and the pop-up player
  • Fixed expand/shrink video player
  • Fixed automatic comments sorting
  • Fixed shorts conversion
  • Fixed missing thumbnail when video is stopped
